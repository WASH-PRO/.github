# WASH-PRO Organization Profile

Этот репозиторий содержит публичный профиль организации **WASH-PRO** на GitHub.

| Файл | Назначение |
| :--- | :--- |
| [`profile/README.md`](profile/README.md) | Отображается на **[github.com/WASH-PRO](https://github.com/WASH-PRO)** во вкладке **Overview** (режим **Public**) |
| [`profile/assets/`](profile/assets/) | Баннер и бейджи из [WASH-PRO-PANEL / main / images](https://github.com/WASH-PRO/WASH-PRO-PANEL/tree/main/images) |

> **Важно:** по [документации GitHub](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile) файл профиля организации **должен** находиться в `profile/README.md`, а не в корне репозитория.

Если профиль не виден — на странице организации выберите **View as → Public** (или **Member**, если настроен `.github-private`).
